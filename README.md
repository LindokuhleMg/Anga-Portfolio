# Anga Siyoyo — Film Director Portfolio

A cinematic filmmaker portfolio website inspired by modern streaming platforms and designed to showcase films, storytelling, and visual artistry.

Built with immersive visuals, smooth animations, and a premium dark aesthetic to create a true cinematic experience.

---

## Features

- Fullscreen cinematic hero video
- Netflix-inspired featured film reel
- Smooth GSAP animations
- Fully responsive design
- Modern cinematic UI/UX
- YouTube film links
- Premium dark theme
- Interactive hover effects

---

## Built With

- HTML5
- CSS3
- JavaScript
- GSAP Animation Library
